# 開発環境構築

Node.jsを導入してください. バージョンは[.tool-versions](../.tool-versions)を参照.

## セットアップ

```shell
$ npm install
```

## コマンド

詳細は[package.json](../package.json)を参照

###　ホットリロード環境で起動

```shell
$ npm run dev
```

### ビルド

```shell
$ npm run build
```

### lint

```shell
$ npm run lint
```

### 型チェック

```shell
$ npm run type-check
```
